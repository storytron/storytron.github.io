INCLINATION EQUATIONS

Inclination equations are used to determine how to branch during play from the currently selected storylet when more than one storylet could be a valid destination. Follow the instructions below to create up to five individual inclination equations for the currently selected storylet.

Select the variables that you would like to use in all of the inclination equations you create (the VARiable columns in the table below will be populated automatically with the variables you select).

1ST VARiable: 2ND VARiable: 3RD VARiable:

Select a storylet from the row's dropdown menu to create an inclination equation. Then enter values for the A, B, and C co-efficients in their respective columns.

Each inclination equation follows this order of operations when calculating during play.

          ( A x 1ST VARiable ) + ( B x 2ND VARiable ) + ( C x 3RD VARiable )

 B 2ND VAR
 C 3RD VAR

During play, the value of each inclination equation will be calculated using the current variable values and their co-efficients. The storylet whose inclination equation has the highest calculated value will be displayed next.
